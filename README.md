# technozoneitcare.com

## Install


```sh
npm install --global gatsby-cli

git clone https://github.com/roottraveller/technozoneitcare.git

cd technozoneitcare
npm install
gatsby develop

gatsby build 
gatsby serve
npm run deploy   
```
